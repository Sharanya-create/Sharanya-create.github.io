Web scraper website and py softwares

Web scraping is an important concept of data science and being a data science enthusiast, i learnt web scraping from coursera and built a web scraper website that interacts with scraper api and gets the web data after scraping it. I have written programs with and without interacting with API, working on terminal interface, and converted it to executable python softwares using pyinstaller. I have built the website using html5,css,js and python-flask.

As web scraping can be used for illegal reasons i have not pushed it on github if interested to view the project then drop me a mail
