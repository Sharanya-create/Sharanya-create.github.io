Developed using java swings and applets

Registration form is similar to the online registration forms or google registration forms. Though not similar to google registration form in its UI but is in functionality as java swings do not allow to provide a similar UI and even I have replaced the submit button by register, to give a better UI as its developed on swings. After registering using the form,the data gets writen on to a local file, here i have not used any ajax call or may be writing to excel sheet. Digital type writer is developed using java applets and I call it so because, just like any conventional type writer it allows typing and saves it on to a destination local file. The only difference here is, being digital,this type writer do not allow typing on paper but on the screen of the computer and its having advantage of saving that info on hard disk of computer in file format. Whereas in the usual type writers we have to keep the papers safe and ensure that they dont get lost. Even in hard disks there is chance of data loss due to system failure but managing this is more easier than managing papers. It also serves as a simple note maker.
